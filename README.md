# TestCafe Boilerplate

TestCafe end-to-end test automation boilerplate, Using Commonjs, Chai and Assert.

## Getting Started

    	1. git clone git@github.com:e2e-boilerplate/testcafe-commonjs-chai-assert.git
    	2. cd testcafe-commonjs-chai-assert
    	3. npm install
    	4. npm run test
